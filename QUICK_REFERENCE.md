# 🎨 PrintScreenApp - 快速参考

## ⚡ 快速开始

```
按 Alt+Q → 拖拽选择区域 → 自动打开编辑器 → 标注 → 点击保存
```

## 🛠️ 工具快捷键

| 按键 | 工具 | 用途 |
|------|------|------|
| **1** | 🖊️ 画笔 | 自由绘制 |
| **2** | ➜ 箭头 | 指向标注 |
| **3** | ▓ 马赛克 | 隐私保护 |
| **4** | ⬜ 矩形 | 矩形框 |
| **5** | ⭕ 圆形 | 圆形框 |

## 🎮 编辑控制

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | ↶ 撤销 |
| `Ctrl+Y` | ↷ 重做 |
| `Esc` | 取消编辑 |
| `Color按钮` | 选择颜色 |
| `Size滑块` | 调整粗细 |

## 💾 操作

- **保存** ✓ - 保存编辑后的截图
- **取消** ✕ - 放弃编辑，保存原始截图

## 📌 使用场景

### 场景1: 强调要点
```
1. 按 1 选择画笔
2. 在关键区域绘制
3. 保存
```

### 场景2: 指示方向
```
1. 按 2 选择箭头
2. 绘制指向箭头
3. 保存
```

### 场景3: 隐藏敏感信息
```
1. 按 3 选择马赛克
2. 涂抹敏感区域
3. 保存
```

### 场景4: 圈出内容
```
1. 按 4 或 5 选择矩形/圆形
2. 框出内容
3. 保存
```

## 🎨 颜色和大小

- **颜色** - 点击工具栏的"Color"按钮打开颜色选择器
- **大小** - 拖动"Size"滑块调整笔画粗细 (1-15)

## 🔄 编辑流程

```
选择工具 → 设置颜色 → 调整大小 → 绘制 → [撤销/重做] → 保存
```

## 💡 提示

✓ 可以多次切换工具进行复杂标注
✓ 所有操作都支持撤销，无需担心错误
✓ 编辑不会影响原始截图
✓ 支持多次保存相同操作

## ❌ 常见错误

| 问题 | 解决方案 |
|------|---------|
| 编辑器不打开 | 确认截图区域有效（宽高都 > 0） |
| 标注不显示 | 检查颜色是否与背景相同 |
| 操作卡顿 | 尝试较小的截图区域 |
| 快捷键无响应 | 确保编辑器窗口获得焦点 |

# 🚀 PrintScreenApp v2.0 - 快速开始

## ⚡ 3秒快速开始

```
1. 按 Alt+Q
2. 拖拽选择区域
3. 编辑器打开 → 点击工具 → 绘制 → 保存
```

## 🎨 5种工具一览

| 按键 | 工具名 | 快捷键 | 功能 |
|------|--------|--------|------|
| 🖊️ | 画笔 | 1 | 自由绘制任意线条 |
| ➜ | 箭头 | 2 | 绘制指向箭头 |
| ▓ | 马赛克 | 3 | 隐藏敏感信息 |
| ⬜ | 矩形 | 4 | 框出重要内容 |
| ⭕ | 圆形 | 5 | 圆形框标注 |

## 🎮 编辑控制快速参考

```
选择工具:     按 1-5 或点击工具按钮
改变颜色:     点击 Color 按钮
调整大小:     拖动 Size 滑块
撤销操作:     按 Ctrl+Z 或点击 ↶ 按钮
恢复操作:     按 Ctrl+Y 或点击 ↷ 按钮
取消编辑:     按 Esc 或点击 ✕ 按钮
保存图像:     点击 ✓ 按钮
```

## 📸 使用场景示例

### 场景1: 突出重点

## 💡 技巧和窍门

### ✓ 可以这样做
- 使用多个工具进行复杂标注
- 随时撤销错误的操作
- 更改颜色和大小直到满意
- 在多个区域使用同一工具

### ✗ 不要这样做
- 不要忘记按 Save 保存 (否则编辑会丢失)
- 不要使用不支持的快捷键
- 不要在编辑器外点击窗口 (需要重新打开)

## 🎨 颜色设置技巧

```
默认颜色: 红色
改变方法: 在编辑器中点击 "Color" 按钮
应用对象: 画笔、箭头、矩形、圆形
马赛克:   颜色不适用 (总是灰色预览)
```

## 📏 大小调节技巧

```
范围: 1-15 像素
默认: 2 像素
调节: 拖动工具栏的 "Size" 滑块
应用: 画笔、箭头、矩形、圆形
提示: 大小越大，标注越明显
```

## 🔄 撤销和重做

```
撤销最后操作:    Ctrl+Z 或 ↶ 按钮
恢复上一操作:    Ctrl+Y 或 ↷ 按钮
支持次数:        无限次
清空重做栈:      进行新操作时自动清空
```

## 🆘 常见问题快速解答

**Q: 编辑器为什么没打开?**  
A: 检查是否成功选择了截图区域 (宽度和高度都要 > 0)

**Q: 标注看不见?**  
A: 检查颜色是否与背景相同，或增加笔画大小

**Q: 怎样才能恢复误删除的标注?**  
A: 按 Ctrl+Y 或点击 ↷ 按钮重做

**Q: 保存后图像去哪里了?**  
A: 保存在剪贴板中，可用 Ctrl+V 粘贴到其他应用

**Q: 能否修改已保存的图像?**  
A: 需要重新截图并编辑 (保存后无法继续编辑)

## 🎯 第一次使用步骤

```
步骤 1: 启动应用
       运行 PrintScreenApp.exe

步骤 2: 触发截图
       按 Alt+Q 或点击"区域截图"按钮

步骤 3: 选择区域
       在屏幕上拖拽选择要截图的区域

步骤 4: 编辑器弹出
       自动打开编辑界面

步骤 5: 选择工具
       按 1-5 或点击工具按钮

步骤 6: 进行标注
       - 点击 Color 选择颜色
       - 拖动 Size 调整大小
       - 在图像上绘制标注

步骤 7: 保存
       点击 ✓ Save 保存图像
       或点击 ✕ Cancel 放弃编辑

步骤 8: 使用
       Ctrl+V 粘贴到其他应用中使用
```

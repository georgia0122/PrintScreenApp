## ✨ 用户功能体验

### 工作流程
```
1. 按 Alt+Q 启动截图
2. 拖拽选择区域
3. 编辑器自动打开 ✨
4. 选择工具进行标注
5. 点击保存
6. 图像保存到剪贴板
```

### 快捷方式
- **数字键 1-5** - 快速切换工具
- **Ctrl+Z/Y** - 撤销/重做
- **Esc** - 取消编辑

### 颜色和大小
- **Color按钮** - 打开颜色选择器
- **Size滑块** - 调整笔画粗细 (1-15)

## 🚀 部署指南

### 系统要求
- Windows 10/11
- .NET 8.0 Runtime

### 构建和运行
```bash
# 克隆/获取代码
cd PrintScreenApp

# 恢复依赖
dotnet restore

# 编译项目
dotnet build

# 运行应用
dotnet run
```

### 发布
```bash
# 生成独立发布包
dotnet publish -c Release -r win-x64 --self-contained
```
